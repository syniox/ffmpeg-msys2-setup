### ffmpeg-setup

**WIP**

A script to download and compile ffmpeg with simplicity and lightweight in mind.  

Dependencies:

```
autoconf automake make cmake pkg-config nasm libtool
```

dav1d needs `meson ninja`

ffplay needs `libsdl2-dev`

External libraries:

* libx264
* libx265
* libdav1d
* libfdk-aac

System libraries:

* libopus
* libvmaf
